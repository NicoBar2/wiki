Datos Abstractos

¿Qué son los datos abstractos?

Los datos abstractos son una estructura de datos que se define en términos de operaciones en lugar de representaciones de datos concretos. En otras palabras, se definen por su comportamiento en lugar de su implementación.

Ejemplos de datos abstractos

  • Listas
  • Colas
  • Pilas
  • Árboles
  • Grafos

Implementaciones de datos abstractos

Los datos abstractos se pueden implementar de diferentes maneras, dependiendo de las necesidades del programa. Algunas implementaciones comunes incluyen:

  • Arreglos
  • Listas enlazadas
  • Punteros

Ejemplo de código en C++

				
class Pila {
public:
    Pila();
    void push(int valor);
    int pop();
    bool isEmpty();
private:
    int datos[100];
    int tope;
};
